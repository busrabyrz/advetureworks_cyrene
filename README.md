🚴‍♀️ Cyrene Data Collective – AdventureWorks Analytics Project

📖 About the Project

This repository contains the visual outputs of the Power BI analytics project developed by the Cyrene Data Collective team.

The project was built on the AdventureWorks dataset and was designed not merely as a reporting exercise, but as a strategic analytics initiative.

Our goal was not to create a dashboard.
Our goal was to build a data-driven decision framework.

🎯 Project Objective

Using the AdventureWorks dataset, we aimed to:

  -  Analyze high-value customer behavior

  -  Identify revenue and profitability drivers

  -  Evaluate product and regional performance

  -  Translate machine learning outputs into business insights

  -  Convert analytical findings into strategic decisions

This project goes beyond visualization.
It represents a structured approach to business intelligence and decision support.

📊 Report Structure

The Power BI report is built on a multi-layered analytical architecture:

  1️⃣ Executive Summary
  Core KPIs including revenue, profitability, customer distribution, and growth trends.

  2️⃣ Customer & Market Analysis
  Demographic structure, income levels, and behavioral patterns.

  3️⃣ Product & Pricing Analysis
  Category-based performance and margin contribution analysis.

  4️⃣ Regional Performance
  Geographical distribution of sales and profitability.

  5️⃣ Strategic Insight Layer
  Machine learning outputs translated into business decisions and connected to product strategy.

🧠 Analytical Approach

  -  SQL-based data modeling
  -  Data cleaning and transformation
  -  Power BI data model design
  -  Advanced DAX measure development
  -  KPI architecture creation
  -  Data storytelling
  -  Converting analytical outputs into strategic decisions

We did not stop at analysis.
We operationalized insights.
